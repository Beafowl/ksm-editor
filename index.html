KSM Chart Editor — Help

Getting started

  1. 📂 Folder opens a song folder (chart + audio auto-load, saving writes back). Or Open .ksh + 🎵 Audio separately, or drag & drop files.
  2. For a new song: 🎵 Audio, then New, set BPM and Offset (ms position in the audio where beat 1 lands), then chart away.
  3. 💾 Save writes the .ksh (or downloads it if no folder is open).

Placing objects

  • BT / FX tool: click a lane = chip · click-drag upward = hold · click an existing chip = remove it.
  • Laser tools: each click adds a point. Click on the same row = slam. Enter/Esc/double-click finishes the laser, right-click removes the last point.
  • Right-click (any tool) deletes what's under the cursor.
  • Select tool: drag notes to move, drag a hold's tail to resize, drag laser points freely.

Effects

  • Select an FX hold → pick effect + parameter in the Inspector (Retrigger;8, Gate;16, Flanger, …).
  • Select a laser → toggle 2× wide (laserrange) and set the filter type (peak / lpf1 / hpf1 / bitc).

Navigation

  • Space play / pause
  • Scroll step by snap · Ctrl+Scroll lane speed
  • ↑/↓ step by snap · PgUp/PgDn measure
  • Home/End start / last note
  • Click / drag the bottom timeline to jump anywhere

Tools

  • 1 Select · 2 BT · 3 FX · 4 Laser L · 5 Laser R

Editing

  • Ctrl+Z undo · Ctrl+Y redo · Del delete selection · Ctrl+S save

Notes

  • Lane spins and unknown lines (zoom, tilt, …) in loaded charts are preserved on save.
  • FX/laser effects are saved to the chart; the editor previews them visually, not audibly.
