KSM Chart Editor — Help

Getting started

  1. 📂 Folder opens a song folder (chart + audio auto-load, saving writes back). Or Open .ksh + 🎵 Audio separately, or drag & drop files.
  2. For a new song: 🎵 Audio, then New, set BPM and Offset (ms position in the audio where beat 1 lands), then chart away.
  3. 💾 Save writes the .ksh (or downloads it if no folder is open).

Placing objects

  • BT / FX tool: click a lane = chip · click-drag upward = hold · click an existing chip = remove it.
  • Laser tools: each click adds a point. Click on the same row = slam. Enter/Esc/double-click finishes the laser, right-click removes the last point. Placing or dragging a point outside the track automatically makes the laser 2× wide (laserrange).
  • Curving: select a laser point → pick a shape (smooth / ease in / ease out) → Curve to next point subdivides the span into an eased curve.
  • Right-click (any tool) deletes what's under the cursor.
  • Select tool: drag notes to move, drag a hold's tail to resize, drag laser points freely. Shift+click selects multiple objects — drag moves them together, Del deletes all, and picking an effect applies to every selected FX hold.

Effects & events

  • Select an FX hold → pick effect + parameter in the Inspector (Retrigger;8, Gate;16, Flanger, …). Custom #define_fx effects from the chart appear too.
  • Select a laser → toggle 2× wide (laserrange) and set the filter type (peak / lpf1 / hpf1 / bitc / custom).
  • Select a laser point → add a lane spin / half-spin / swing with a length (place them on slams).
  • +BPM / +Sig insert tempo & time-signature changes at the cursor; +Cmd inserts raw lines (zoom_top, tilt, stop, …).
  • The Events panel lists everything in the current measure — delete with ×.

Navigation

  • Space play / pause
  • Scroll step by snap · Ctrl+Scroll lane speed
  • ↑/↓ step by snap · PgUp/PgDn measure
  • Home/End start / last note
  • Click / drag the bottom timeline to jump anywhere

Game view

  • Tab (or the View dropdown) switches Editor / Split / Game view. Split is the default.
  • The game view renders the track in 3D with USC's camera: zoom_top/bottom/side, tilt (laser + manual), spins, swings and stops all animate. Scrub through a spin to preview it.
  • Lane Speed (numeric, any value) applies to both views; Ctrl+Scroll on either view adjusts it.

Tools

  • 1 Select · 2 BT · 3 FX · 4 Laser L · 5 Laser R

Editing

  • Ctrl+Z undo · Ctrl+Y redo · Del delete selection · Ctrl+S save

Notes

  • Unknown chart lines are preserved verbatim on save.
  • FX preview plays an approximation of hold effects and the laser filter during playback (toggle in the Playback panel). The game's own DSP will sound richer.
  • Song, hitsound and metronome volumes are independent sliders.

Chart events reference

Everything the Events panel can show. Insert with +Cmd / “+ Add event”; camera values interpolate linearly between two keyframes of the same command.

t=185BPM (tempo) change.
beat=7/8Time signature — sets the measure length (use +Sig).
filtertype=peakFilter sound while a laser moves: peak (wah), lpf1 (low-pass), hpf1 (high-pass), bitc (bitcrush) or a #define_filter name. Set per-laser via the Inspector.
zoom_top=50Camera pitch — tips the far end of the track down (+) or up (−).
zoom_bottom=-75Camera zoom — pulls the camera away (−) or pushes it in (+).
zoom_side=30Camera shift — slides the track sideways.
tilt=biggerLaser tilt strength: normal / bigger / biggest / zero; keep_* holds the current tilt.
tilt=1Manual roll: 1 ≈ −10°, interpolates to the next numeric tilt.
stop=192Freezes scrolling for N/192 of a measure (192 = one 4/4 measure) while the music continues.
@(192 …Lane spin on a slam (set on a laser point in the Inspector): @(/@) full spin, @</@> half spin, S</S> swing; number = duration in 1/192s of a measure.
laserrange_l=2xWide (2×) laser — Inspector toggle on a selected laser.
fx-l=Retrigger;8Effect on an FX hold — set via the Inspector.
fx-l_se=clap.wavCustom hit sample for FX chips on that side.
center_split=100Splits the track down the middle, pushing the halves apart.
lane_toggle=48Fades the track out/in over N/192 of a measure.
