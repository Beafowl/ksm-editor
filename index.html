KSM Chart Editor — Help

Getting started

  1. 📂 Open folder (launch screen or File menu) opens a song folder — chart + audio auto-load and saving writes back. You can also drag & drop .ksh / audio files anywhere.
  2. For a new song: ✚ New chart (launch screen or File menu) — pick the audio, then the Setup page opens for metadata and timing.
  3. ⚙ Setup (T): one page for metadata (title, artist, difficulty, …) and timing — set BPM and Offset directly, or play the song and tap T to the beat to estimate both.
  4. 📁 File ▾: Save .ksh writes the chart (or downloads it if no folder is open); Save .kson exports the new KSM v2 format; Save folder as .zip downloads the whole song folder as an archive.
  5. The chart list next to Save switches between the folder's difficulties — ＋ New .ksh… adds a new one that shares the song's audio and timing.

Placing objects

  • BT / FX tool: click a lane = chip · click-drag upward = hold · click an existing chip = remove it.
  • Laser tools: each click adds a point. Click on the same row = slam. Clicking the end point of an existing laser continues that laser. Enter/Esc/double-click finishes the laser, right-click removes the last point. Placing or dragging a point outside the track automatically makes the laser 2× wide (laserrange).
  • Curving: Select tool → Shift+click two or more points per laser (several lasers at once work too) → 〜 Spline through points replaces each laser between its selected points with a smooth curve through them. Density is automatic — as dense as possible without creating slams, so spans narrower than 14 ticks stay straight.
  • Right-click (any tool) deletes what's under the cursor — or use the Remove tool (6) to delete with plain clicks.
  • Select tool: drag notes to move, drag a hold's tail to resize, drag laser points freely. Shift+click selects multiple objects — drag moves them together, Del deletes all, and picking an effect applies to every selected FX hold. When laser points overlap, clicking the same spot again picks the next stacked point.

Effects & events

  • Select an FX hold → pick effect + parameter in the Inspector (Retrigger;8, Gate;16, Flanger, …). Custom #define_fx effects from the chart appear too.
  • Select a laser → toggle 2× wide (laserrange) and set the filter type (peak / lpf1 / hpf1 / bitc / custom).
  • Select a laser point → add a lane spin / half-spin / swing with a length (place them on slams).
  • + Add event at cursor (Events panel) opens a form listing every event type — tempo, time signature, camera, tilt, stop, … — with the matching input and an explanation of what it does. “Custom command” takes any raw line.
  • The Events panel lists everything in the current measure — delete with ×.

Navigation

  • Space play / pause
  • Scroll step by snap · Ctrl+Scroll lane speed
  • ↑/↓ step by snap · PgUp/PgDn measure
  • Home/End start / last note
  • Click / drag the bottom timeline to jump anywhere
  • B add/remove a bookmark at the cursor (or add a named one via + Add event) · ,/. jump to previous/next bookmark (rename via the Events panel; saved as // comments)

Game view

  • Tab (or the View dropdown) switches Editor / Split / Game view. Split is the default.
  • The game view renders the track in 3D with USC's camera: zoom_top/bottom/side, tilt (laser + manual), spins, swings and stops all animate. Scrub through a spin to preview it.
  • Editor and game lane speeds are set separately in 🎚 Preferences; Ctrl+Scroll on a view adjusts that view's speed.

Tools

  • 1 Select · 2 BT · 3 FX · 4 Laser L · 5 Laser R · 6 Remove

Editing

  • Ctrl+Z undo · Ctrl+Y redo · Del delete selection · Ctrl+S save

Notes

  • Unknown chart lines are preserved verbatim on save.
  • Playback previews an approximation of FX-hold effects and the laser filter — the game's own DSP will sound richer. Assist tick (Preferences) toggles the note ticks; laser slams always play.
  • 🎚 Preferences holds snap, lane speeds, playback speed and volumes. Slowing playback keeps the song's pitch.
  • ✨ ChartGPT generates charts: pick level and skill sliders (0–1); with a song loaded it follows the audio (Audio strength sets how tightly). Range “Between bookmarks” regenerates only that section of the current chart. The model ships with the editor.

Chart events reference

Everything the Events panel can show. Insert with + Add event at cursor; camera values interpolate linearly between two keyframes of the same command.

t=185BPM (tempo) change.
beat=7/8Time signature — sets the measure length, applied from the current measure.
filtertype=peakFilter sound while a laser moves: peak (wah), lpf1 (low-pass), hpf1 (high-pass), bitc (bitcrush) or a #define_filter name. Set per-laser via the Inspector.
zoom_top=50Camera pitch — tips the far end of the track down (+) or up (−).
zoom_bottom=-75Camera zoom — pulls the camera away (−) or pushes it in (+).
zoom_side=30Camera shift — slides the track sideways.
tilt=biggerLaser tilt strength: normal / bigger / biggest / zero; keep_* holds the current tilt.
tilt=1Manual roll: 1 ≈ −10°, interpolates to the next numeric tilt.
stop=192Freezes scrolling for N/192 of a measure (192 = one 4/4 measure) while the music continues.
@(192 …Lane spin on a slam (set on a laser point in the Inspector): @(/@) full spin, @</@> half spin, S</S> swing; number = duration in 1/192s of a measure.
laserrange_l=2xWide (2×) laser — Inspector toggle on a selected laser.
fx-l=Retrigger;8Effect on an FX hold — set via the Inspector.
fx-l_se=clap.wavCustom hit sample for FX chips on that side.
center_split=100Splits the track down the middle, pushing the halves apart.
lane_toggle=48Fades the track out/in over N/192 of a measure.
